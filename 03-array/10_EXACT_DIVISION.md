รับเลขจำนวนเต็มบวกจำนวน 5 จำนวน และ แสดงผลลัพธ์ จำนวนที่ หารด้วย 10 ลงตัว โดยถ้ามีมากกว่า 1 ค่า จะ ขั้นด้วยเครื่องหมาย , (comma) ระหว่างตัวเลข

และถ้า ไม่มีจำนวนใดๆ ที่หาร10ลงตัว ให้แสดงประโยค “Not found.”

Test Case 1

- input :
```
> Enter number 1 :1
> Enter number 2 :2
> Enter number 3 :3
> Enter number 4 :4
> Enter number 5 :5
```


- Output :
```
> Not found.
```

Test Case 2

- Input :
```
> Enter number 1 :10
> Enter number 2 :2
> Enter number 3 :30
> Enter number 4 :40
> Enter number 5 :6
```
- Output :
```
> 10,30,40
```
